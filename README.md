# nerm-portfolio

This is a practice project for my portfolio. It is a NERM stack app.

## Project Structure

The project structure is as follows:

```
nerm-portfolio/
├── client/        // Vite.js
│   ├── src/
│   │   ├── main.js
│   │   └── ...
│   ├── vite.config.js
│   └── ...
├── server/        // Express.js
│   ├── index.js
│   └── ...
├── package.json
└── ...
```

## Tech stack
* Vite.js
* React.js
* Express.js
* Jest.js
